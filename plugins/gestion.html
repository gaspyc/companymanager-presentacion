La Realidad de tu Negocio

Una Biblioteca, Configuraciones Distintas

Adquirís el plugin una sola vez y queda disponible para todas tus sedes. Pero cada sucursal decide si lo activa y con qué parámetros opera.

La sede mayorista usa el plugin de precios con lista mayorista. La minorista usa el mismo plugin pero con reglas de descuento por volumen. Son configuraciones distintas del mismo plugin.

Dónde se Configura

Ajustes de Sede: El Panel de Control

Cada sucursal tiene su propia pantalla de Ajustes. Ahí encontrás los slots por rol: uno para facturación, uno para pagos, uno para precios. Cada slot acepta el plugin que corresponde a ese rol.

Si el slot está vacío, esa funcionalidad no está activa en esa sede. Simple.

Mismo Plugin, Distinta Config

Parámetros Propios de Cada Sede

Cuando activás un plugin en una sede, configurás sus parámetros para ese local. El plugin de precios en la sucursal centro tiene un porcentaje de descuento distinto al del depósito mayorista.

Los parámetros viven en el espacio aislado de la sede. Cambiar la config de un local nunca afecta al otro.

El Principio Central

Cada Sede es un Mundo Aparte

Lo que pase en una sede no contamina a las demás. Un plugin roto en el depósito no afecta al e-commerce. Una configuración mal puesta en el centro no toca al mayorista.
