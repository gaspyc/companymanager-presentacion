Logo circular
superpuesto.

El banner no es una sola imagen. El local puede subir una versión de escritorio y una de mobile con sus propios puntos de recorte. buildBannerImageUrl recibe la variante 'desktop' o 'mobile' según isMobileViewport, que un listener de resize actualiza en vivo.

banner_crops por variante

El objeto banner_crops tiene claves desktop y mobile con coordenadas de recorte. Si no existe, usa la imagen original.

Logo en margen negativo

El logo tiene margin-top: -44px (desktop) o -45px (mobile) para montarse sobre el borde del banner. El borde blanco de 3px crea la separación visual.

Fallbacks en cascada

Banner: buildBannerImageUrl → banner_url → imagen de Unsplash. Logo: buildPublicImageUrl → ícono fa-user-circle.

El Widget de Horarios

Siempre muestra
el estado de hoy.
El resto, al expandir.

El horario del local llega como JSON o string desde el perfil. El computed scheduleDisplayText busca el día actual con new Date().getDay() y muestra el estado puntual en el badge. Un click expande el dropdown con la semana completa.

Hoy abierto

todaySchedule.isOpen = true → muestra "Hoy: 09:00 - 18:00".

Hoy cerrado

isOpen = false → muestra "Hoy: Cerrado".

Parsing defensivo

Si schedule es string, se intenta JSON.parse. Si falla o no es array, se muestra el texto plano sin dropdown.

Cierre al click exterior

closeSchedule se registra en document vía onMounted y se limpia en onUnmounted. Si el click fue dentro del ref del dropdown, no cierra.

Acciones y Contenido

Todo lo que el local
quiere que el cliente haga.

WhatsApp directo

Construye https://wa.me/{cleanPhone} limpiando todos los caracteres no numéricos con .replace(/\D/g, ''). Se abre en la app nativa del teléfono.

Portal de empleos

El componente JobApplicationBanner solo se muestra si profile.is_hiring = true. Integra el modal de postulación sin código extra en StoreProfile.

¿Cómo comprar?

Abre StoreHowToBuyModal con el prop hidePrices calculado desde la config del storefront. StoreProfile lo resuelve en onMounted.

Redes sociales

parsedSocialLinks maneja string JSON y objeto nativo. Solo se muestran las redes que tengan valor no vacío.

Dirección simplificada

simplifiedAddress detecta el formato "número, calle, zona" y lo reorganiza como "calle número, zona". Más legible que la dirección completa de la API.

Descripción expandible

Si el texto supera 140 caracteres, se trunca con -webkit-line-clamp: 4 y aparece el botón "Ver más/menos". En desktop está siempre expandida.
